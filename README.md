# Transactions gateway

To run app:
```shell
make run
```

To generate swagger docs (127.0.0.1:8080/swagger):
```shell
make gen-docs
```

To test:
```shell
make test
```

To test with verbose output:
```shell
make test-verbose
```

To run linter:
```shell
make lint
```

To run linter with verbose output:
```shell
make lint-verbose
```
